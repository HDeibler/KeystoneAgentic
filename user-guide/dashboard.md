# Dashboard Overview

The **Dashboard** is your home base in KeystoneAgentic. It provides a quick overview of your organization's status, recent activity, and shortcuts to common tasks.

---

## Dashboard Layout

When you log in, you'll see four main sections:

### 1. Welcome Banner
A personalized greeting with your name and organization. This confirms you're logged in to the correct organization.

### 2. Quick Actions Grid
Fast access to the most common tasks:

**New Chat** (Coming Soon)
- Start a conversation with an AI model
- Choose from your synced models
- Access your knowledge base automatically

**Browse Models**
- Discover available AI models
- Filter by capabilities, price, and provider
- Sync new models to your organization

**Upload Documents**
- Add files to your knowledge base
- Build your RAG library
- Support for PDF, Word, text, and more

**View Analytics** (Admin only)
- See detailed usage statistics
- Track token consumption by user and model
- Monitor conversation activity

### Quick Links
- **Upgrade Plan**: Increase your quota and user seats
- **Manage Billing**: View invoices and update payment methods
- **View Settings**: Access organization settings

## Statistics Overview

Real-time metrics for your organization:

### User Seats
- **Current / Total**: Number of active users vs. plan limit
- **Example**: "3 / 5 seats" means 3 members, 2 available
- **Color Coding**:
  - Green: Under 80% utilization
  - Yellow: 80-100% utilization
  - Red: At limit

### Monthly Tokens
- **Used / Allocated**: Token consumption vs. monthly quota
- **Percentage Bar**: Visual representation of usage
- **Reset Date**: When your quota refreshes
- **Color Coding**:
  - Green: Under 80% usage
  - Yellow: 80-100% usage
  - Red: Over 100% (grace period up to 110%)

### Requests This Month
- **Total Count**: Number of AI requests made
- **Average per Day**: Daily request rate
- **Trend**: Up or down compared to previous month

### Active Conversations
- **Open Chats**: Ongoing conversation sessions
- **Recent Activity**: Last active conversation timestamp

## Dashboard for Different Roles

What you see depends on your role:

### Admin
- Full dashboard with all widgets
- Access to analytics and usage data
- Billing and subscription information
- Team activity and management options

### Member
- Dashboard with activity and models
- No billing information
- Can see team roster
- Limited organization settings

### Viewer
- Read-only dashboard
- Can view statistics and activity
- Cannot see billing details
- No management actions available

[Learn more about roles and permissions](/docs/organization/roles-permissions)

## Tips for Dashboard Efficiency

### Check Your Quota Regularly
- Glance at the **Monthly Tokens** widget daily
- Set a reminder when approaching 80% usage
- Plan upgrades before hitting limits

### Monitor Team Activity
- Review **User Seats** to know when you're near capacity
- Watch **Recent Activity** to understand team usage patterns
- Identify power users who may need additional training

### Use Quick Actions
- Bookmark frequently used features
- Quick Actions save clicks compared to sidebar navigation
- Keyboard shortcuts coming soon

### Stay Informed
- Check the dashboard at the start of each work session
- Review trends weekly to optimize usage
- Act on warnings and notifications promptly

## Dashboard Updates & Refresh

### Real-Time Updates
- Statistics refresh every 60 seconds automatically
- Recent activity updates when new events occur
- No manual refresh needed

### Manual Refresh
- Click the refresh icon (top-right) to force update
- Useful after syncing models or uploading documents
- Clears cached data

### Network Issues
- If data doesn't load, check your internet connection
- Dashboard shows "Loading..." state during fetch
- Error messages appear if fetch fails
